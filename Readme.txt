How to get started?
1. Install necessary libraries
    a. Go the directory
    b. use the following command
        i. `pip3 install pandas`
        ii. `pip3 install numpy`
        iii. `pip3 install scikit-learn`
2. Drop your csv file in the folder
3. Change the filename in the variable 'filename' in main.py
4. Give the name of your <to be predicted> column name to the variable 'target_column'
5. Run the main.py file

Go thorugh QuantumSolver_docs.py to understand how to use the quantum code on an example
